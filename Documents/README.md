# FluentMetric AI Documentation

> Index of indexes. Pick your audience tree below.

FluentMetric AI is a Salesforce-native dashboard app that makes Einstein
Generative AI Audit & Feedback data humanly readable. Two editions ship
from this repo (Lightning + Tableau Next) — repo-level overview and
edition matrix in [../README.md](../README.md).

## Three audience trees

### 🛠️ [Admin/](Admin/) — installing or operating the package

UI-first runbooks for Salesforce admins. Prerequisites, install via 2GP
URL, configuration, troubleshooting, upgrades. **No CLI required.**

Start at [Admin/README.md](Admin/README.md).

### 💻 [Developer/](Developer/) — extending or building on the package

Local-dev setup, Apex / LWC / DMO contracts, coding standards, release
engineering. Authoritative live DMO schema lives here.

Start at [Developer/README.md](Developer/README.md).

### 🏛️ [Architect/](Architect/) — design rationale and decisions

System architecture, edition selection guide, ADRs (Architecture Decision
Records). Original design intent and trade-off analyses.

Start at [Architect/README.md](Architect/README.md).
